# project overview
A live cycle rental business operated at BIT Mesra for hostel students.
The project captures real booking, revenue, repeat usage, feedback, and seasonal demand data generated during daily operations (June–July 2025).
 A real cycle rental business operated at BIT Mesra for hostel students. The rentals mainly served Hostel 4 (≈90%), with a few from H10 & H11.
Data reflects realistic bookings, credits, repeat customers, cancellations, and seasonal demand spikes during June–July 2025.

# Business Context

Primary demand from Hostel 4 (~90%), with limited usage from H10 and H11

Bookings via QR, WhatsApp, walk-in, and website

Revenue of ~₹10K during peak summer months

# Dataset

Rental transactions (date, duration, revenue)

Customer type (new vs repeat)

Booking channel

Feedback and ratings

Hostel-wise demand distribution


# Analysis & KPIs

Fleet utilization and idle time

Daily and weekly rental trends

Repeat customer behavior

Hostel-wise demand concentration

Booking channel contribution

# Key Insights

Demand highly concentrated in a single hostel, indicating location-driven utilization

Weekday usage dominated rentals, suggesting academic schedule influence

Repeat customers contributed significantly to stable demand
